My code for MITx: 6.00.1x Introduction to Computer Science and Programming Using Python

## Prerequisites
* [Python](https://www.python.org/)
* [pipenv](https://github.com/pypa/pipenv)

### Download
```
git clone
cd mit-intro-comp-sci
pipenv install
```

### Run tests
```
pytest
```

### Run tests in watch mode:
```
ptw
```
